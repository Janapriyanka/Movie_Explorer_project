# Movie_Explorer_project
Movie Explorer Project

A responsive movie browsing website built using HTML, CSS, and JavaScript.
This project allows users to explore movies, view collections, and interact with a clean and modern UI.

🚀 Features
🎥 Homepage with movie highlights
📂 Movie collections page with search & filter
📩 Contact page with form validation
📱 Fully responsive design
🌐 Multi-page navigation system
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Tailwind CSS
